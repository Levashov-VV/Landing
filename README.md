# Клуб четырёх коней

Адаптивный лендинг для шахматного турнира, выполненный как тестовый задание на чистом HTML, CSS и JavaScript.

## О проекте

Проект представляет собой одностраничный промо-сайт с несколькими смысловыми секциями, адаптивной вёрсткой и интерактивными элементами.

Реализованы:

- hero-секция со сложной декоративной композицией;
- бегущая строка;
- блок с описанием события и адаптивной таблицей;
- секция этапов
- секция участников с бесконечной каруселью;

## Стек

- HTML5
- CSS3
- Vanilla JavaScript
